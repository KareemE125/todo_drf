# todo_drf
Django Rest Framework TODO api

### Include
1. Api Endpoints
2. CRUD Operations
3. Exception Handler Wrapper (decorator)
4. Authentication
5. JWT tokens
6. Token middleware check

## Endpoints
```
# Todos
GET api/todos
POST api/todos/create
GET api/todos/:id
PUT api/todos/:id/update
DELETE api/todos/:id/delete

# Auth
POST api/auth/login
POST api/auth/signup
```
